🔥 Configuração Firebase

Como obter a configuração:

  1. Acesse console.firebase.google.com e crie/abra um projeto
  2. No menu lateral: Firestore Database → Criar banco de dados
  3. Nas configurações do projeto (⚙️) → Seus aplicativos → Web (</>)
  4. Copie o objeto firebaseConfig (só o JSON entre { })
  5. Cole no campo abaixo e clique em Salvar e conectar
⚠️ Regras do Firestore — para uso interno, configure assim em Firestore → Regras:
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /{document=**} {
      allow read, write: if true;
    }
  }
}
